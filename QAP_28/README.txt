 Итоговый проект 28.1 по автомаизации тестирования.
 Сделан под windows так как как в основном все описание и примеры использовались в windows
 разбираться как это реальзиванно в linux нет просто физического времени. Работа.
 Так же изменены логин, пароль, почта для входа. Раскрывать оригиналы не горю желаением. 
 Выполнена проверка авторизации сайта https://b2c.passport.rt.ru
 в requirements.txt рекомендованные расширерия для установки.
 Для запуска тестов необходимо ввести в терминал pytest -v --driver Chrome --driver-path /путь нахождения драйвера/chromedriver.exe tests_RT_tabs.py.
